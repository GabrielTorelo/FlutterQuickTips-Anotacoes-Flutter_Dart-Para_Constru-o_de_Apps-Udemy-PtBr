O que é Flutter?
  Resposta: Flutter é um framework para criação de aplicativos nativos para iOS, Android, web e desktop a partir de uma única base de código.
  Ou seja: O flutter permite aos desenvolvedores criarem aplicativos móveis com uma única linguagem de programação (Dart) e uma única base de código, simplificando o desenvolvimento e a manutenção dos aplicativos.

Características do Flutter:
  * Tudo é Widget:
    Ou seja: No Flutter, tudo que você vê na tela é um widget, desde layouts e botões até animações e estilos.
  * Suporte para diferentes plataformas (iOS e Android):
    Ou seja: Com o Flutter, é possível criar aplicativos que funcionam em várias plataformas (iOS e Android) sem a necessidade de escrever código específico para cada um.
  * Hot Reload:
    Ou seja: Flutter oferece a funcionalidade de "hot reload", permitindo que os desenvolvedores vejam instantaneamente as mudanças feitas no código sem a necessidade de recompilar o aplicativo inteiro.
  * Comunidade Ativa e Suporte do Google:
    Ou seja: Flutter é suportado por uma comunidade ativa de desenvolvedores e pelo Google, que oferece documentação abrangente, tutoriais e pacotes adicionais para expandir as capacidades do Flutter.
  * Integração Fácil com APIs:
    Ou seja: Flutter facilita a integração com APIs e serviços backend, permitindo a construção de aplicativos ricos em funcionalidades que se conectam a serviços externos de forma eficiente.

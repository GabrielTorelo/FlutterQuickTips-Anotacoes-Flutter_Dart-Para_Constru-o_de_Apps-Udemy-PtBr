Introdução as variáveis:
  Conceito: Variáveis são espaços de memória que reservamos para armazenar valores.
  Tipagem: Dart é uma linguagem fortemente tipada.
    Ou seja: É sempre necessário declarar o tipo de uma variável.
  Exemplos:
    1 - Variáveis do tipo inteiro:
      int idade = 24;
    2 - Variáveis do tipo double:
      double altura = 1.80;
    3 - Variáveis do tipo String:
      String nome = "Gabriel";
    4 - Variáveis do tipo boolean:
      bool maiorDeIdade = true;
    5 - Variáveis do tipo dynamic (pode mudar o tipo da variável a qualquer momento):
      dynamic variavel = 1;
